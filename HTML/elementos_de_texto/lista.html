Tipo de lista em Html

Lista ordenadas

As listas ordenadas usam números para listar os itens

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista não ordenadas

AS listas não ordenadas usam marcadores para listar os

  • Maça
  • Banana
  • Laranja

Listas de de definição são usadas para apresentar termos e suas descrições

Html
Linguagem de marcação de hiperTexto usada para criar páginas na web
Css
Folhas de estilo em cascata, usadas para estilizar a aparerencia de elemento HTML
Javascript
Linguagem de programação utilizada para tornar as páginas ativas
